The most efficient reactor doesn't produce the most power. To get a efficient reactor that can still generate lots of power, you should optimize for power while limiting the cell count using the “Max Allowed” box for []. To get the ideal cell count, divide the fuel's base processing time by the time you want a fuel to last. Enabling symmetry and/or limiting block counts often speeds up the algorithm. The algorithm uses fixed seed: different runs will always yield the same results unless there is an update to the algorithm (last update: 28/7/2025).
You can use Forgematica to automate the building of your reactor. To do so, download the schematic and put it in the schematics folder of your Minecraft instance. Note that Forgematica also needs MaFgLib to run.
Alternatively, you could also use Building Gadgets to automate the building of your reactor by copying the BG String and pasting it in the Template Manager to create a template.
